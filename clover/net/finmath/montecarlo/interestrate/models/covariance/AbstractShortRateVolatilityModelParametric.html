38    /**
39    * Base class for parametric volatility models, see also {@link AbstractShortRateVolatilityModel}.
40    *
41    * Parametric models feature a parameter vector which can be inspected
42    * and modified for calibration purposes.
43    *
44    * The parameter vector may have zero length, which indicated that the model
45    * is not calibrateable.
46    *
47    * This class includes the implementation of a generic calibration algorithm.
48    * If you provide an arbitrary list of calibration products, the class can return
49    * a new instance where the parameters are chosen such that the (weighted) root-mean-square
50    * error of the difference of the value of the calibration products and given target
51    * values is minimized.
52    *
53    * @author Ruben Duarte

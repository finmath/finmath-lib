15    /**
16    * Implements the volatility model
17    * \[
18    * \sigma_{i}(t_{j}) = ( a + b (T_{i}-t_{j}) ) exp(-c (T_{i}-t_{j})) + d \text{.}
19    * \]
20    *
21    * The parameters here have some interpretation:
22    * <ul>
23    * <li>The parameter a: an initial volatility level.</li>
24    * <li>The parameter b: the slope at the short end (shortly before maturity).</li>
25    * <li>The parameter c: exponential decay of the volatility in time-to-maturity.</li>
26    * <li>The parameter d: if c &gt; 0 this is the very long term volatility level.</li>
27    * </ul>
28    *
29    * Note that this model results in a terminal (Black 76) volatility which is given
30    * by
31    * \[
32    * \left( \sigma^{\text{Black}}_{i}(t_{k}) \right)^2 = \frac{1}{t_{k}} \sum_{j=0}^{k-1} \left( ( a + b (T_{i}-t_{j}) ) exp(-c (T_{i}-t_{j})) + d \right)^{2} (t_{j+1}-t_{j})
33    * \]
34    * i.e., the instantaneous volatility is given by the picewise constant approximation of the function
35    * \[
36    * \sigma_{i}(t) = ( a + b (T_{i}-t) ) exp(-c (T_{i}-t)) + d
37    * \]
38    * on the time discretization \( \{ t_{j} \} \). For the exact integration of this formula see {@link LIBORVolatilityModelFourParameterExponentialFormIntegrated}.
39    *
40    * @author Christian Fries
41    * @version 1.0
42    */

19    * \]
20    *
21    * The parameters here have some interpretation:
22    * <ul>
23    * <li>The parameter a: an initial volatility level.</li>
24    * <li>The parameter b: the slope at the short end (shortly before maturity).</li>
25    * <li>The parameter c: exponential decay of the volatility in time-to-maturity.</li>
26    * <li>The parameter d: if c &gt; 0 this is the very long term volatility level.</li>
27    * </ul>
28    *
29    * Note that this model results in a terminal (Black 76) volatility which is given
30    * by
31    * \[
32    * \left( \sigma^{\text{Black}}_{i}(t_{k}) \right)^2 = \frac{1}{t_{k} \int_{0}^{t_{k}} \left( ( a + b (T_{i}-t) ) exp(-c (T_{i}-t)) + d \right)^{2} \ \mathrm{d}t \text{.}

34    *

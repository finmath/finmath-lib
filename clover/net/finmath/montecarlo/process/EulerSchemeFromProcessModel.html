22    /**
23    * This class implements some numerical schemes for multi-dimensional multi-factor Ito process.
24    *
25    * It features the standard Euler scheme and the standard predictor-corrector Euler scheme
26    * for <i>Y</i>, then applies the <i>state space transform</i> \( X = f(Y) \). For the standard Euler scheme
27    * the process Y is discretized as
28    * \[
29    * Y(t_{i+1}) = Y(t_{i}) + \mu(t_{i}) \Delta t_{i} + \sigma(t_{i}) \Delta W(t_{i}) \text{.}
30    * \]
31   
32    *
33    * Hence, using the <i>state space transform</i>, it is possible to create a log-Euler scheme, i.e.,
34    * \[
35    * X(t_{i+1}) = X(t_{i}) \cdot \exp\left( (\mu(t_{i}) - \frac{1}{2} \sigma(t_{i})^2) \Delta t_{i} + \sigma(t_{i}) \Delta W(t_{i}) \right) \text{.}
36    * \]
37    *
38    * The dimension is called <code>numberOfComponents</code> here. The default for <code>numberOfFactors</code> is 1.
39    *
40    * @author Christian Fries
41    * @see MonteCarloProcess The interface definition contains more details.
42    * @version 1.4
43    */

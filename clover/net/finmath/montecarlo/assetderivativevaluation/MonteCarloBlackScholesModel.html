21    /**
22    * This class glues together a <code>BlackScholeModel</code> and a Monte-Carlo implementation of a <code>MonteCarloProcessFromProcessModel</code>
23    * and forms a Monte-Carlo implementation of the Black-Scholes Model by implementing <code>AssetModelMonteCarloSimulationModel</code>.
24    *
25    * The model is
26    * \[
27    * dS = r S dt + \sigma S dW, \quad S(0) = S_{0},
28    * \]
29    * \[
30    * dN = r N dt, \quad N(0) = N_{0},
31    * \]
32    *
33    * The class provides the model of S to an <code>{@link net.finmath.montecarlo.process.MonteCarloProcess}</code> via the specification of
34    * \( f = exp \), \( \mu = r - \frac{1}{2} \sigma^2 \), \( \lambda_{1,1} = \sigma \), i.e.,
35    * of the SDE
36    * \[
37    * dX = \mu dt + \lambda_{1,1} dW, \quad X(0) = \log(S_{0}),
38    * \]
39    * with \( S = f(X) \). See {@link net.finmath.montecarlo.process.MonteCarloProcess} for the notation.
40    *
41    * @author Christian Fries
42    * @see net.finmath.montecarlo.process.MonteCarloProcess The interface for numerical schemes.
43    * @see net.finmath.montecarlo.model.ProcessModel The interface for models provinding parameters to numerical schemes.
44    * @version 1.0
45    */

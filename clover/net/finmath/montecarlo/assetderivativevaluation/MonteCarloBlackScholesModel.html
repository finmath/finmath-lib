23    /**
24    * This class glues together a <code>BlackScholeModel</code> and a Monte-Carlo implementation of a <code>MonteCarloProcessFromProcessModel</code>
25    * and forms a Monte-Carlo implementation of the Black-Scholes Model by implementing <code>AssetModelMonteCarloSimulationModel</code>.
26    *
27    * The model is
28    * \[
29    * dS = r S dt + \sigma S dW, \quad S(0) = S_{0},
30    * \]
31    * \[
32    * dN = r N dt, \quad N(0) = N_{0},
33    * \]
34    *
35    * The class provides the model of S to an <code>{@link net.finmath.montecarlo.process.MonteCarloProcess}</code> via the specification of
36    * \( f = exp \), \( \mu = r - \frac{1}{2} \sigma^2 \), \( \lambda_{1,1} = \sigma \), i.e.,
37    * of the SDE
38    * \[
39    * dX = \mu dt + \lambda_{1,1} dW, \quad X(0) = \log(S_{0}),
40    * \]
41    * with \( S = f(X) \). See {@link net.finmath.montecarlo.process.MonteCarloProcess} for the notation.
42    *
43    * @author Christian Fries
44    * @see net.finmath.montecarlo.process.MonteCarloProcess The interface for numerical schemes.
45    * @see net.finmath.montecarlo.model.ProcessModel The interface for models provinding parameters to numerical schemes.
46    * @version 1.0
47    */

16    /**
17    * This class implements an inhomogeneous displaced log-normal model, that is, it provides the drift and volatility specification
18    * and performs the calculation of the numeraire (consistent with the dynamics, i.e. the drift).
19    *
20    * The model is
21    * \[
22    * \mathrm{d}S = r S dt + \sigma (S + d) \mathrm{d}W, \quad S(0) = S_{0},
23    * \]
24    * \[
25    * \mathrm{d}N = r N \mathrm{d}t, \quad N(0) = N_{0},
26    * \]
27    *
28    * Note that
29    * \[
30    * \mathrm{d}(S/N) = \sigma (S/N + d/N) \mathrm{d}W
31    * \]
32    * i.e.
33    * \[
34    * \mathrm{d}(S/N + d/N) = -r d/N dt + \sigma (S/N + d/N) \mathrm{d}W
35    * \]
36    *
37    * The class provides the model of S to an <code>{@link net.finmath.montecarlo.process.MonteCarloProcess}</code>
38    * via the specification of (via X = S/N+d/N)
39    * \( S = f(X) = N X - d \), \( \mu = d \frac{exp(- r t_2) - exp(- r t_1)}{t_2-t_1} \), \( \lambda_{1,1} = \sigma X \),
40    * of the SDE
41    * \[
42    * dX = \mu dt + \lambda_{1,1} dW, \quad X(0) = S_{0} + e,
43    * \]
44    * with \( N(0) = 1 \). See {@link net.finmath.montecarlo.process.MonteCarloProcess} for the notation.
45    *
46    * @author Christian Fries
47    * @see net.finmath.montecarlo.process.MonteCarloProcess The interface for numerical schemes.
48    * @see net.finmath.montecarlo.model.ProcessModel The interface for models provinding parameters to numerical schemes.
49    * @version 1.0
50    */

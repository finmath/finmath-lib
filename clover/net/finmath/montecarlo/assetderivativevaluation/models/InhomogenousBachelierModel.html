17    /**
18    * This class implements a (variant of the) Bachelier model, that is, it provides the drift and volatility specification
19    * and performs the calculation of the numeraire (consistent with the dynamics, i.e. the drift).
20    *
21    * The model is
22    * \[
23    * dS = r S \mathrm{d}t + \sigma \mathrm{d}W, \quad S(0) = S_{0},
24    * \]
25    * \[
26    * dN = r N \mathrm{d}t, \quad N(0) = N_{0},
27    * \]
28    *
29    * <p>
30    *
31    * Note, the model corresponds to the following dynamic for the numeraire relative stock value \( F(t) = S(t)/N(t) \):
32    * \[
33    * \mathrm{d}F = exp(-r t) \sigma \mathrm{d}W, \quad F(0) = F_{0} = S_{0}/N_{0},
34    * \]
35    *
36    * <p>
37    *
38    * Note that the variance of \( \int_{t}^{t+\Delta t} \mathrm{d}F \) is
39    * \[ \int_{t}^{t+\Delta t} ( exp(-r s) \sigma )^{2} \mathrm{d}s = \frac{\exp(-2 r t)}{2 r} \left( 1 - \exp(-2 r \Delta t) \right) \]
40    *
41    *
42    * The class provides the model of S to an <code>{@link net.finmath.montecarlo.process.MonteCarloProcess}</code> via the specification of
43    * \( f = \text{identity} \), \( \mu = \frac{exp(r \Delta t_{i}) - 1}{\Delta t_{i}} S(t_{i}) \), \( \lambda_{1,1} = \sigma \frac{exp(-2 r t_{i}) - exp(-2 r t_{i+1})}{2 r \Delta t_{i}} \), i.e.,
44    * of the SDE
45    * \[
46    * dX = \mu dt + \lambda_{1,1} dW, \quad X(0) = \log(S_{0}),
47    * \]
48    * with \( S = X \). See {@link net.finmath.montecarlo.process.MonteCarloProcess} for the notation.
49    *
50    * The model's implied Bachelier volatility for a given maturity T is
51    * <code>volatility * Math.sqrt((Math.exp(2 * riskFreeRate * optionMaturity) - 1)/(2*riskFreeRate*optionMaturity))</code>
52    *
53    * @author Christian Fries
54    * @see net.finmath.montecarlo.process.MonteCarloProcess The interface for numerical schemes.
55    * @see net.finmath.montecarlo.model.ProcessModel The interface for models provinding parameters to numerical schemes.
56    * @version 1.0
57    */

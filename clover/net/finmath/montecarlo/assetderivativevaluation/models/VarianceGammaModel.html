15    /**
16    * This class implements a <i>Variance Gamma Model</i>, that is, it provides the drift and volatility specification
17    * and performs the calculation of the numeraire (consistent with the dynamics, i.e. the drift).
18    *
19    * The model is
20    * \[
21    * dS_t = r S dt + S dL, \quad S(0) = S_{0},
22    * \]
23    * \[
24    * dN = r N dt, \quad N(0) = N_{0},
25    * \]
26    *
27    * where the process L is a <code>{@link net.finmath.montecarlo.VarianceGammaProcess}</code>.
28    *
29    * @author Alessandro Gnoatto
30    * @see net.finmath.montecarlo.process.MonteCarloProcess The interface for numerical schemes.

32    * @version 1.0
33    */

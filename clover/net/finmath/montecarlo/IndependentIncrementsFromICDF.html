18    * Implementation of a time-discrete n-dimensional sequence of independent increments
19    * <i>W = (W<sub>1</sub>,...,W<sub>n</sub>)</i> form a given set of inverse
20    * cumulative distribution functions.
21    *
22    * Independent increments is a sequence of independent random variables index
23    * by the time index associated with the time discretization. At each time step
24    * the increment is a d-dimensional random variable \( Z(t_{i}) \), where d is <code>numberOfFactors</code>.
25    * where each component of \( Z_{j}(t_{i}) \) is given by
26    * \[
27    * Z_{j}(t_{i}) = ICDF_{i,j}(U_{i,j})
28    * \]
29    * for a sequence of independent uniform distributes random variables U_{i,j}.
30    *
31    * The inverse cumulative distribution functions \( ICDF_{i,j} \) are given by
32    * <code>inverseCumulativeDistributionFunctions</code> as the
33    * map \( i \mapsto ( j \mapsto ICDF_{i,j} ) \) (here i is the time index and j is the factor (component).
34    *
35    * Each \( U_{i,j} \) is samples using <code>numberOfPaths</code>.
36    *
37    * The class is immutable and thread safe. It uses lazy initialization.
38    *
39    * @author Christian Fries
40    * @version 1.6
41    */
